# 音樂播放器期末作業
### 簡介  
* 根據整個學期老師上課所教的與課本知識結合所製作的期末專題作業  
* 可將Music_Player.apk直接安裝至android虛擬器上操作
### 技術介紹  
* 簡單的Intent意圖切換頁面  
* 多媒體應用  
* SQLite資料庫應用
